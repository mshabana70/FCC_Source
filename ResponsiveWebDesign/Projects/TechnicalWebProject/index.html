CSS Flexbox

  • Basic Concept:

    The Flexibile Box Module (also know as 'flexbox'), is a one dimensional layout model. This module is used as a method for space distribution between webpage elements in a user interface, and also has powerful item alignment capabilities.

    When we say 'One Dimensional,' what we mean by this is that Flexbox handles layouts in one dimension at a time. With flexbox, you are creating a layout either as a row or as a column, never both at the same time. This is different from CSS Grid Layout, which is capable of styling layouts for columns and rows similtaneously.

  • Flex Box Axes

    There are two axes that a developer should consider when working with flexbox - the main axis and the cross axis.

    • The Main Axis:

      The main axes is defined by the flex-direction, which has four possible values: row, row-reverse, column, column-reverse.
